Whitelisted Domains

Manage the list of public domains for pageload monitoring

If you share pageload monitoring data with us, by default the names of the sites you visit are anonymized. This means that we cannot know which sites you visit or if you and another Fathom user have visited the same sites. However, in order to be able to study the network performance across users and various networks, we DO NOT anonymize the identify of a small set of popular domains listed below.

You can remove (uncheck the box) any domain names that you feel unconfortable about sharing with us below. Don't forget to save changes. The changes will only affect future data collection.
